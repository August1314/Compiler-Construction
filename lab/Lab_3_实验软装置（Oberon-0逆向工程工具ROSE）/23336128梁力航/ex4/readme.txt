三、实现说明
------------
1. 采用递归下降预测分析技术，根据 Oberon-0 的翻译模式
   手工编写语法分析和语法制导翻译程序
2. 语法分析过程中同时生成流程图（Flowchart），集成 ROSE
   软装置的 Flowchart API
3. 对文法进行了必要转换：消除左递归、提取左公因子
4. 每一非终结符对应一个递归子程序，开始符号对应主程序
5. 继承属性用形式参数表示，综合属性用返回值表示

四、分析策略比较
----------------
通过实验三（JavaCUP 自底向上）和实验四（递归下降自顶向下）
的实践对比：
- 递归下降更直观、易调试，方法直接对应非终结符
- LALR 更通用，能处理更复杂的文法
- 递归下降便于在任意位置嵌入语义动作
- 两种方法在 Oberon-0 这种简单语言上效果相近
